1. What is Normalization? 
---> Normalization бһӮбһәбһҮбһ¶бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҡбҹҖбһ”бһ…бҹҶбһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ Database
бһҠбһҫбһҳбҹ’бһ”бһёбһҖбһ¶бһҸбҹӢбһ”бһ“бҹ’бһҗбһҷбһ—бһ¶бһ–бһҹбҹ’бһ‘бһҪбһ“ (Redundancy) бһ“бһ·бһ„бһҡбһҖбҹ’бһҹбһ¶бһ“бһјбһңбһ—бһ¶бһ–бһҸбҹ’бһҡбһ№бһҳбһҸбҹ’бһҡбһјбһңбһ“бҹғбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ (Data Integrity)бҹ”

2. Step of Normalization? 
---> In Normalization having 3 step for follow up : 
    - 1NF
    - 2NF
    - 3NF

3. Why using Normalization??
---> We use Normalization for : 
    - бһҖбһ¶бһҸбҹӢбһ”бһ“бҹ’бһҗбһҷбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҹбҹ’бһ‘бһҪбһ“
    - бһҮбҹҖбһҹбһңбһ¶бһ„ Data Anomaly (бһ”бһүбҹ’бһ бһ¶бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷ) : бһҳбһ·бһ“бһўбһ¶бһ…бһ”бһүбҹ’бһ…бһјбһӣбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҗбҹ’бһҳбһёбһ”бһ¶бһ“ бһ”бһҫбһҒбҹ’бһңбҹҮбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһ•бҹ’бһҹбҹҒбһ„, бһҖбҹӮбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҳбһҪбһҷ бһҸбҹ’бһҡбһјбһңбһҖбҹӮбһ…бҹ’бһҡбһҫбһ“бһҖбһ“бҹ’бһӣбҹӮбһ„ вҶ’ бһ„бһ¶бһҷбһҒбһ»бһҹ,бһӣбһ»бһ”бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҳбһҪбһҷ вҶ’ бһ”бһ¶бһҸбҹӢбһ–бҹҗбһҸбҹҢбһҳбһ¶бһ“бһҹбҹҶбһҒбһ¶бһ“бҹӢбһ•бҹ’бһҹбҹҒбһ„бһ‘бҹҖбһҸ
    - бһ„бһ¶бһҷбһҹбҹ’бһҡбһҪбһӣбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһӮбҹ’бһҡбһ”бҹӢбһӮбҹ’бһҡбһ„ Database
    - using in real system

4. About rule of 1NF(First Normal Form)? 
---> бһӣбһҖбҹ’бһҒбһҺбҹҮбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһ’бҹ’бһңбһҫбһҡ 1NF (First Normal Form) : 
    ~ бһ бһ¶бһҳ Field бһ¬вҖӢ ColumnбһҠбҹӮбһӣбһ”бҹ’бһҡбһҫбһ”бһјбһҖбһ”бһүбҹ’бһ…бһјбһӣбһӮбҹ’бһ“бһ¶ бһ¬бһҖбҹҸбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ cell бһҳбһҪбһҷбһҳбһ¶бһ“бһҸбһҳбҹ’бһӣбҹғбһҸбҹӮбһҳбһҪбһҷбһ”бҹүбһ»бһҺбҹ’бһҺбҹ„бҹҮ
    ~ Field бһҳбһҪбһҷбҹ—бһҳбһ·бһ“бһІбҹ’бһҷбһҳбһ¶бһ“бһҲбҹ’бһҳбҹ„бҹҮбһҮбһ¶бһ“бҹӢбһӮбҹ’бһ“бһ¶ бһ¬бһҖбһёбһӮбҹ’бһҳбһ¶бһ“ Column бһҹбҹ’бһ‘бһҪбһ“
    ~ Field бһҳбһҪбһҷбһҸбҹ’бһҡбһјбһңбһҳбһ¶бһ“ DataType бһҸбҹӮбһҳбһҪбһҷбһӮбһҸбҹӢ
    ~ Data бһҠбҹӮбһӣбһҳбһ·бһ“бһҹбҹҶбһҒбһ¶бһ“бҹӢ бһ¬бһҳбһ·бһ“бһ–бһ¶бһҖбҹӢбһ–бҹҗбһ“бҹ’бһ’бһҹбҹ„бҹҮбһҳбһ·бһ“бһ…бһ¶бҹҶбһ”бһ¶бһ…бҹӢбһ•бҹ’бһ‘бһ»бһҖбҹ’бһ“бһ»бһ„ Table бһ‘бҹҒ
    ~ бһҸбҹ’бһҡбһјбһңбһҳбһ¶бһ“ column бһҳбһҪбһҷбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ identify row uniquely (Id)

5. About Rule of 2NF(Second Normal Form)?
---> бһӣбһҖбҹ’бһҒбһҺбҹҮбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһ’бҹ’бһңбһҫбһҡ 2NF (Second Normal Form) : 
    ~ бһҖбҹҶбһҺбһҸбҹӢбһҖбҹ’бһҡбһ»бһҳбһҡбһ”бһҹбҹӢ Field бһҠбҹӮбһӣбһҹбҹ’бһ‘бһҪбһ“
    ~ бһӣбһ»бһ”бһҖбҹ’бһҡбһ»бһҳвҖӢ Field бһҠбҹӮбһӣбһҹбҹ’бһ‘бһҪбһ“бһ бһҫбһҷбһ”бҹӮбһ„бһ…бҹӮбһҖ Table
    ~ бһҖбҹҶбһҺбһҸбҹӢбһҖбҹ’бһҡбһ»бһҳ Field бһҠбҹӮбһӣбһҳбһ·бһ“бһўбһ¶бһҹбҹ’бһҡбҹҗбһҷ Table бһ…бҹҒбһү
    ~ бһҹбҹ’бһңбҹӮбһ„бһҡбһҖ Table бһҺбһ¶бһҠбҹӮбһӣбһўбһ¶бһҹбҹ’бһҡбҹҗбһҷбһ”бҹ’бһҡбһҹбһ·бһ“бһ”бһҫбһӮбҹ’бһҳбһ¶бһ“бһ‘бҹҒбһ”бһ„бҹ’бһҖбһҫбһҸ вҖӢTable бһҗбҹ’бһҳбһё
    ~ Primary Key бһ–бһё table бһҳбҹҒбһҸбҹ’бһҡбһјбһңбһ”бҹ’бһҡбһҫбһҮбһ¶ Foreign Key бһҖбҹ’бһ“бһ»бһ„ table бһ•бҹ’бһҹбҹҒбһ„

6. About Rule of 3NF(Third Normal Form)?
---> бһӣбһҖбҹ’бһҒбһҺбҹҮбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһ’бҹ’бһңбһҫбһҡ 3NF (Third Normal Form) : 
    ~ бһҸбҹ’бһҡбһјбһңбһҸбҹӮбһҮбһ¶бһ”бҹӢбһҖбһҳбҹ’бһҡбһ·бһҸ 2NF бһҮбһ¶бһҳбһ»бһ“бһҹбһ·бһ“
    ~ бһҠбһ¶бһҖбҹӢбһҲбҹ’бһҳбҹ„бҹҮбһІбҹ’бһҷ table бһ‘бҹ…бһҸбһ¶бһҳбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҠбҹӮбһӣ Store бһҮбһ¶бһҹбһҳбҹ’бһӮбһ¶бһӣбҹӢ
    ~ бһӮбҹ’бһҳбһ¶бһ“ Transitive Dependency
