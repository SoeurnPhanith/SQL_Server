1. What is Store Procedure?
---> Store Procedure គឺជា​​​​ SQL command មួយក្រុម ដែលយើង save នៅក្នុង database
ហើយអាច call ប្រើម្តងហើយម្តងទៀត ដែលវាមានលក្ខណះស្រដៀងទៅនឹង function ក្នុង​Programmingដែរ​​។

2. Why using Store Procedure?
---> We use store Procedure Because : 
    - Reuseable of query
    - good speed and performance 
    - security not expose table directly
    - Easy to update and maintaince

3. Syntax to create store Procedure? 
---> CREATE PROCEDURE ProcedureName
     AS
     BEGIN
        -- SQL Query here
     END;    

4. When we use store Procedure? 
    *Modify Data : Insert / Update / Delete → SP can do it safely
    *Multiple Steps / Business Logic  : Example: Insert order + update stock + log activity
    *Transactions : BEGIN, COMMIT, ROLLBACK → SP handles it
    *Security / Permission Control  : User executes SP but cannot access tables directly    
    *Reuse Logic / Maintenance : Centralized code → easy to update, use in multiple applications
